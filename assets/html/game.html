Emoji Game

How To Play Emoji Game?

1

Start the Game

Click on the "Start Game" button, either on the main page or within your user account.

2

Game Guide

The Emoji Game consists of 8 pairs of emojis that you need to match. Emojis are randomly placed on the screen and revealed for 2 minutes at the beginning.

3

Search and Match

Click on any emoji to reveal it. Match two different emojis to find pairs.

4

Scoring

Display the game start time and remaining time on the screen. Earn points for each correctly matched pair. The goal is to find all pairs within the limited time.

5

End of the Game

If you find all pairs, the game ends, and you are declared the winner. If the limited time expires, the game concludes, and no further moves are allowed.

6

Score Registration

Points earned are added to your user account. Your total game scores are visible in your user profile.
